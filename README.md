# PikaProjBackend

**Self-hosted voice AI backend for Pika iOS (Whisper, Ollama, TTS, voice profiles, auth).**

[![Python](https://img.shields.io/badge/Python-3.x-blue)](https://python.org) [![FastAPI](https://img.shields.io/badge/FastAPI-teal)](https://fastapi.tiangolo.com)

## Key Capabilities
- Google OAuth + sessions.
- Async voice chat jobs.
- XTTS/CosyVoice personalization.
- Cloud Run ready.

Paired with Pika iOS clients.
